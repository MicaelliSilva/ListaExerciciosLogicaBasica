# ListaExerciciosLogicaBasica
Lista de Exercicios de Logica Basica em C

## Declaração de Variáveis
1. Crie um programa que declare três variáveis (int, float e char) e exiba seus valores na tela.
2. Peça ao usuário para digitar sua idade e mostre: 'Você tem X anos'.
## Estruturas Condicionais (if / else)
3. Peça dois números inteiros e mostre qual deles é maior.
4. Leia a nota de um aluno e informe se ele está aprovado (≥ 6.0) ou reprovado (< 6.0).
5. Leia um número e diga se ele é par ou ímpar.
## Laços de Repetição (for / while)
6. Mostre os números de 1 a 10 usando for.
7. Leia um número inteiro e mostre sua tabuada (1 a 10).
8. Some todos os números de 1 a 100 usando while.
9. Leia um número N e calcule o fatorial usando for.
10. Peça números ao usuário até que ele digite 0 (usando while).
## Estrutura Switch
11. Crie um menu com as opções: Somar, Subtrair, Multiplicar, Dividir.
12. Peça um número de 1 a 7 e mostre o dia da semana correspondente.
13. Leia um número de 1 a 12 e exiba o nome do mês correspondente.
## Desafios (Misturando os Conceitos)
14. Crie um programa que peça a idade de 5 pessoas, use for para armazenar e depois mostre a
média das idades.
15. Faça um programa que leia um número inteiro positivo e verifique se ele é número primo.
16. Crie um programa que simule um caixa eletrônico: o usuário digita o valor a sacar e o programa
informa quantas notas de 100, 50, 20, 10, 5 e 2 serão entregues.
17. Desenvolva um programa que peça a senha ao usuário (senha correta = 1234). Enquanto ele
não acertar, o programa deve continuar pedindo.
